# Data Gate Web

Nuxt app and Nitro backend for Data Gate.

The web app owns:

- CSV upload and local dataset storage
- Dataset listing and public metadata responses
- Nitro Tasks for deterministic profiling checks
- Default Quality Gate workflow execution
- Report artifact generation and preview
- Chat UI and high-level Data Gate AI SDK tools

Chat attachments are disabled for the MVP. CSV datasets should be uploaded through the Datasets page so raw data stays local to the server.
